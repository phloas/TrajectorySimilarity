# 轨迹相似度任务研究

## 项目概括
本项目旨在研究轨迹相似度任务。它涵盖了轨迹数据集的预处理、三元组的生成、轨迹之间距离的计算、轨迹图的生成、图特征提取、数据集划分以及模型训练等多个核心模块。
